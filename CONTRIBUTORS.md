## 🤝 How to Contribute

We welcome and encourage open-source contributions! If you would like to help improve **rewind.git**:

1. **Fork the Repository**  
   Create your own fork of `Garvit-821/rewind-git`.
   
2. **Create a Feature Branch**  
   Name it descriptive of the change, e.g., `git checkout -b feature/webgl-renderers`.

3. **Commit your Changes**  
   Write clean, descriptive commit messages:
   ```bash
   git commit -m "feat(canvas): optimize physics repulsion loop for large trees"
   ```

4. **Submit a Pull Request**  
   Push to your fork and submit a PR to the main branch of `Garvit-821/rewind-git`.
